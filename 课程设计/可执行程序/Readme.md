### 程序运行的一些注意事项

* mac 平台开发生成程序格式为unix
* 程序初始化前需要在根目录下导入"jz001.txt"等文档
* 实现用户注册登陆功能需要在根目录下建立users文件夹
* 在users文件夹里先要初始建立"num.txt",并赋值0；
* 路径中存在汉字可能运行失败
